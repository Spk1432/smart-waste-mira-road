# ♻️ Smart Waste Management System — Mira Road
### Mira-Bhayandar Municipal Corporation (MBMC)

A professional, fully functional web application for smart waste management in Mira Road area.

## 🌟 Features
- **Login / Register** — Citizen & Employee roles
- **Google Maps** — Street view + Satellite + Hybrid
- **Bin Map** — 8 real Mira Road locations with live fill levels
- **File Complaints** — With photo upload
- **Dispatch Trucks** — MBMC Employee can dispatch from map
- **Reports & Analytics** — Weekly charts
- **Dark Mode** — Full dark theme support
- **Fully Responsive** — Mobile + Desktop
